Intersection Of Two Sorted Arrays

Introduction

The intersection of two sorted arrays problem involves finding the common elements present in both arrays. Given that the arrays are sorted, we can leverage their sorted order to find the intersection efficiently.
The Two-Pointer technique is particularly effective for this problem because it leverages the sorted nature of the arrays to efficiently find common elements without needing additional space or complex data structures.

Step By Step Explanation

Here's a detailed explanation of the problem, along with an approach to solve it:

  1. Use two index variables i and j, initial values i = 0, j = 0
  2. If arr1[i] is smaller than arr2[j] then print arr1[i] and increment i.
    If arr1[i] is greater than arr2[j] then print arr2[j] and increment j.
  3. Completion: If both are the same size, print any of them and increment both i and j. Print remaining elements of the larger array.

This algorithm ensures efficient intersection finding in linear time, O(m + n), where m and n are the lengths of nums1 and nums2, respectively. It has O(1) additional space complexity as it only requires space for storing the intersection elements.

Python Code

                
    def intersect(nums1, nums2):
    intersection = []
    i, j = 0, 0

    while i < len(nums1) and j < len(nums2):
        if nums1[i] == nums2[j]:
            intersection.append(nums1[i])
            i += 1
            j += 1
        elif nums1[i] < nums2[j]:
            i += 1
        else:
            j += 1

    return intersection

    # Example usage
    nums1 = [1, 2, 2, 3, 4, 5]
    nums2 = [2, 2, 3, 5]
    print(intersect(nums1, nums2))  # Output: [2, 2, 3, 5]

Example

Find the intersection of two sorted arrays OR in other words, given 2 sorted arrays, find all the elements which occur in both arrays.

The image illustrates the process of finding the union and intersection of two sorted arrays, Array 1 and Array 2, using a two-pointer technique.
Initially, the pointers are at the first elements of both arrays.
The smaller value between the two pointed elements (5 from Array 2) is added to the union, and the pointer for Array 2 is incremented. When the values at both pointers are equal (10), the value is added to both the union and intersection, and both pointers are incremented.
The process continues, comparing elements and updating the union and intersection accordingly, until all elements are processed.
