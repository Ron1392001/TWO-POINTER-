Reversing a Linked List

Introduction

Reversing a linked list is a common problem in computer science, especially when dealing with data structures. The goal is to reverse the pointers in the linked list so that the last node becomes the first node, and vice versa.
The Two-Pointer technique is an efficient method to reverse a linked list. This technique involves using two pointers to keep track of the current node and the previous node as we iterate through the list, effectively reversing the links between nodes.

Step By Step Explanation

Here's a detailed explanation of the problem, along with an approach to solve it:

  1. ListNode Class: Defines the structure of a node in a singly linked list, with a value (val) and a pointer to the next node (next).
  2. Initialization: prev is set to None (indicating the end of the reversed list), and curr is set to the head of the list.
  3. While Loop: Continues until curr is None (end of the list). - Store next_temp: Temporarily store the next node (curr.next) so we don't lose the reference.
    - Reverse Pointer: Set curr.next to prev to reverse the pointer.
    - Move Pointers: Advance prev to curr and curr to next_temp.
  4. Return: prev will be the new head of the reversed linked list once the loop finishes.

This approach efficiently reverses the linked list in linear time, O(n), where n is the number of nodes in the list, with O(1) additional space complexity.

Python Code

                
    class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def reverse_linked_list(head):
    prev = None
    curr = head

    while curr:
        # Temporarily store the next node
        next_temp = curr.next
        # Reverse the current node's pointer
        curr.next = prev
        # Move pointers one position ahead
        prev = curr
        curr = next_temp

    # prev will be the new head after the loop ends
    return prev

# Helper function to create a linked list from a list
def create_linked_list(arr):
    if not arr:
        return None
    head = ListNode(arr[0])
    current = head
    for value in arr[1:]:
        current.next = ListNode(value)
        current = current.next
    return head

# Helper function to print a linked list
def print_linked_list(head):
    result = []
    while head:
        result.append(head.val)
        head = head.next
    print(result)

Example usage
arr = [1, 2, 3, 4, 5]
head = create_linked_list(arr)
print("Original Linked List:")
print_linked_list(head)

reversed_head = reverse_linked_list(head)
print("Reversed Linked List:")
print_linked_list(reversed_head)
