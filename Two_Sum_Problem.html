Two Sum Problem

Introduction

Given an array of integers and a target sum, find two numbers in the array that add up to the target sum with using Two-Pointer technique.
The Two-Pointer technique is a powerful tool for solving array-related problems.
However, for the standard Two Sum problem where the order of indices matters and we need to return the original indices, the Two-Pointer technique isn't directly applicable.
Instead, the two-pointer technique is most effective when the array is sorted, and we need to find pairs that sum up to a given target.

Step By Step Explanation

  1. Has Array Two Candidates (A[], ar_size, sum)
  2. Sort the array in non-decreasing order.
  3. Initialize two index variables to find the candidate elements in the sorted array.
  4. Initialize first to the leftmost index: l = 0
  5. Initialize second the rightmost index: r = ar_size-1
  6. Loop while l < r. • If (A[l] + A[r] == sum) then return 1 • Else if( A[l] + A[r] < sum ) then l++ • Else r–
  7. No candidates in the whole array – return 0

This approach ensures the algorithm runs in linear time, O(NlogN), Time complexity for sorting the array making it efficient.

Python Code

                
def hasArrayTwoCandidates(A, arr_size, sum):
    # Sort the elements
    A.sort()

    # Initialize two pointers
    l = 0
    r = arr_size - 1

    # Now look for the two candidates in the sorted array
    while l < r:
        if A[l] + A[r] == sum:
            return True
        elif A[l] + A[r] < sum:
            l += 1
        else:  # A[l] + A[r] > sum
            r -= 1

    return False

# Driver program to test above function
if __name__ == "__main__":
    A = [1, 4, 45, 6, 10, -8]
    n = 16
    arr_size = len(A)

    # Function calling
    if hasArrayTwoCandidates(A, arr_size, n):
        print("Yes")
    else:
        print("No")

Example

Let an array be {1, 4, 45, 6, 10, -8} and sum to find be 16
After sorting the array
A = {-8, 1, 4, 6, 10, 45}
Now, increment ‘l’ when the sum of the pair is less than the required sum and decrement ‘r’ when the sum of the pair is more than the required sum.
This is because when the sum is less than the required sum then to get the number which could increase the sum of pair, start moving from left to right(also sort the array) thus “l++” and vice versa.

Initialize l = 0, r = 5
A[l] + A[r] ( -8 + 45) > 16 => decrement r. Now r = 4
A[l] + A[r] ( -8 + 10) increment l. Now l = 1
A[l] + A[r] ( 1 + 10) increment l. Now l = 2
A[l] + A[r] ( 4 + 10) increment l. Now l = 3
A[l] + A[r] ( 6 + 10) == 16 => Found candidates (return 1)
