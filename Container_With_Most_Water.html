Container With Most Water

Introduction

Given n non-negative integers representing the heights of walls, where the width of each wall is 1, find the maximum area of water that can be trapped between the walls.
The Two-Pointer technique is highly efficient for solving the Container With Most Water problem.
By strategically moving the pointers, we can explore all potential containers and find the one with the maximum area without unnecessary computations.
This approach optimally balances the need to check multiple pairs while minimizing the number of comparisons needed.

Step By Step Explanation

An efficient approach to solve this problem is to use the two-pointer technique. Here’s how it works:

  1. Initialize Two Pointers: Start with two pointers, one at the beginning (left) and one at the end (right) of the array.
  2. Calculate the Area: Compute the area formed by the lines at the two pointers and the x-axis.
  3. Move the Pointers: Move the pointer pointing to the shorter line towards the other pointer, as the height of the container is limited by the shorter line.
    Update the maximum area found so far during each iteration.
  4. Repeat: Continue this process until the two pointers meet.

This approach ensures the algorithm runs in linear time, O(n), Since only one traversal of the array is required.

Python Code

                
def maxContainer(arr):
    i = 0
    j = len(arr) -1
    maxArea = 0

    while i < j:
    # Calculate the area
    # and update the variable maxArea, if greater
        maxArea = max(maxArea, min(arr[i], arr[j]) * (j - i))

        if arr[i] < arr[j]:
            i += 1
        else:
            j -= 1
    return maxArea

Example


Input: array = [1, 5, 4, 3]
Output: 6
Explanation :
5 and 3 are distance 2 apart.
So the size of the base = 2.
Height of container = min(5, 3) = 3.
So total area = 3 * 2 = 6


Input: array = [1, 5, 6, 3, 4, 2]
Output: 12
Explanation :
1 and 4 are distance 3 apart.
Therefore, the size of the base = 3.
Height of the container will be a minimum of (5,4) = 4
Hence, The total area becomes 3∗4 = 12 units.
