Longest Substring

Introduction

The problem of finding the longest substring without repeating characters is a common algorithmic challenge often encountered in coding interviews and competitive programming.
The objective is to identify the longest contiguous subsequence within a given string such that all the characters in that subsequence are unique.
The Two-Pointer technique, is an efficient way to solve this problem. This method uses two pointers to represent the current window of characters being considered, ensuring that all characters in the window are unique.

Step By Step Explanation

follow these steps in order to fine the Longest SubString:

  1. Dictionary char index: This stores the last index of each character encountered in the string.
  2. Pointer Initialization: left starts at the beginning (0) of the string. The right pointer iterates through each character in the string.
  3. Duplicate Character Handling: If the character at right is found in char_index and its stored index is greater than or equal to left, it means there's a duplicate within the current window. In this case, move the left pointer to one position right of the last occurrence of the duplicate character.
  4. Update char index: Store/update the current index of the character at right.
  5. Pointer Movement: The pointers are moved towards each other if the characters match.
  6. Calculate Window Size: The size of the current window is right - left + 1. Update max_length if the current window size is larger.
  7. Completion: Continue the process until the right pointer has traversed the entire string.

This algorithm ensures that each character is processed at most twice (once by the right pointer and potentially once by the left pointer), resulting in a linear time complexity of O(n), where n is the length of the string. This makes it efficient for solving the problem.

Python Code

                
def length_of_longest_substring(s):
char_index = {}
left = 0
max_length = 0

for right in range(len(s)):
    if s[right] in char_index and char_index[s[right]] >= left:
        # Move the left pointer to the right of the last occurrence of the
        # duplicate character
        left = char_index[s[right]] + 1

    # Update the last occurrence of the current character
    char_index[s[right]] = right

    # Calculate the current window size and update max_length if it's larger
    current_length = right - left + 1
    max_length = max(max_length, current_length)

return max_length
                

# Example usage
s = "abcabcbb"
print(length_of_longest_substring(s))  # Output: 3 ("abc")

s = "bbbbb"
print(length_of_longest_substring(s))  # Output: 1 ("b")

s = "pwwkew"
print(length_of_longest_substring(s))  # Output: 3 ("wke")

Example

Run a loop from i = 0 till N – 1 and consider a visited array.
Run a nested loop from j = i + 1 to N – 1 and check whether the current character S[j] has already been visited.
If true, break from the loop and consider the next window.
Else, mark the current character as visited and maximize the length as j – i + 1.
Remove the first character of the previous window and continue.
