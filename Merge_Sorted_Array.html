Merge Sorted Arrays

Introduction

Merge sort is a sorting algorithm that follows the divide-and-conquer approach. It works by recursively dividing the input array into smaller subarrays and sorting those subarrays then merging them back together to obtain the sorted array.
The Two-Pointer technique provides an efficient way to merge two sorted arrays.
This method ensures that the merging process is done in a single pass through each array, making it both time and space efficient.

Step By Step Explanation

  1. Start with two pointers, one for each array. These pointers will help traverse the arrays.
  2. Compare the current elements pointed to by the pointers.
  3. Append the smaller element to the result array and move the pointer forward in the array from which the element was taken.
  4. Continue the comparison and appending process until one of the arrays is fully traversed.
  5. If there are remaining elements in either array after the main loop, append all remaining elements to the result array.

This approach ensures the algorithm runs in linear time, O(n1 + n2), where n1 and n2 are the lengths of the two arrays

Python Code

                
def mergeArrays(arr1, arr2, n1, n2):
    arr3 = [None] * (n1 + n2)
    i = 0
    j = 0
    k = 0

    # Traverse both arrays
    while i < n1 and j < n2:
        # Check if current element of the first array is smaller than the current element of
        # the second array. If yes, store the first array element and increment the first array
        # index. Otherwise, do the same with the second array.
        if arr1[i] < arr2[j]:
            arr3[k] = arr1[i]
            k = k + 1
            i = i + 1
        else:
            arr3[k] = arr2[j]
            k = k + 1
            j = j + 1

    # Store remaining elements of the first array
    while i < n1:
        arr3[k] = arr1[i]
        k = k + 1
        i = i + 1

    # Store remaining elements of the second array
    while j < n2:
        arr3[k] = arr2[j]
        k = k + 1
        j = j + 1

    return arr3

# Example usage
arr1 = [1, 3, 5]
arr2 = [2, 4, 6]
n1 = len(arr1)
n2 = len(arr2)

merged_array = mergeArrays(arr1, arr2, n1, n2)

print("Array after merging")
for i in range(n1 + n2):
    print(str(merged_array[i]), end=" ")

Example

The example demonstrates merging two sorted arrays arr1 ([5, 8, 9]) and arr2 ([4, 7, 8]) into a sorted array arr3. Initially, pointers i and j compare elements from both arrays and place the smaller one into arr3, advancing the pointer of the array from which the element was taken. Once all elements from arr2 are merged, the remaining elements of arr1 are copied directly to arr3, resulting in a merged sorted array [4, 5, 7, 8, 8, 9].


The first While loop breaks , Second While loop copies all elements from arr1 to arr3


The algorithm uses a two-pointer technique to traverse both arrays and merge them in sorted order. The merged array is constructed by comparing elements from both arrays and inserting the smaller element into the merged array. After one of the arrays is fully traversed, the remaining elements from the other array are copied to the merged array.
