Remove Duplicates from Sorted Array

Introduction

Given a sorted array, remove the duplicates in-place such that each element appears only once and return the new length.
The Two-Pointer technique is an efficient way to solve this problem, leveraging the sorted nature of the array.
By maintaining two pointers, we can efficiently identify and remove duplicates while preserving the relative order of the elements.

Step By Step Explanation

  1. Initialize Pointers: Use one pointer to iterate through the array (i), and another pointer to mark the position of the last unique element (j).
  2. Iterate and Compare: Start iterating the array from the second element. Compare each element with the element at j.
  3. Update and Move: If a new unique element is found, increment j and update the position j with this new element.
  4. Return: Finally, j + 1 will be the number of unique elements.

This approach ensures the algorithm runs in linear time, O(n), since the array is loop once.

Python Code

                
def remove_duplicates(arr):
    n = len(arr)
    if n == 0 or n == 1:
        return n

    # To store index of next unique element
    j = 0

    # Doing same as done in Method 1
    # Just maintaining another updated index i.e. j
    for i in range(n - 1):
        if arr[i] != arr[i + 1]:
            arr[j] = arr[i]
            j += 1

    arr[j] = arr[n - 1]

    return j + 1

Example

Input: arr[] = {1, 2, 2, 3, 4, 4, 4, 5, 5}
Output: arr[] = {1, 2, 3, 4, 5}

Original array:
{1, 1, 2, 3, 4, 4, 5, 6, 6, 6}
After removing duplicates from the above sorted array:
{1, 2, 3, 4, 5, 6}
