Move Zeroes

Introduction

In the Move Zeroes problem, we are given an array and we have to move all the zeroes to the end of the array, while the non-zero elements should maintain the respective order.

The Two-Pointer technique is an efficient way to solve the Move Zeroes problem.
By strategically using two pointers, we can rearrange the elements in-place without requiring additional space, making this approach both time and space efficient.

Step By Step Explanation

  1. Iterate the array from 0 to N.
  2. Keep two pointers, one for zero elements and other for non-zero elements.
  3. Swap every zero elements with the non-zero element that comes just after it.

This approach ensures the algorithm runs in linear time, O(n), since the array is loop once.

Python Code

                
    def moveZerosToEnd(nums):
        j = 0

        for i in range(len(nums)):
            if nums[i]!= 0 and nums[j]== 0:
                nums[i], nums[j]= nums[j], nums[i]
            if nums[j]!= 0:
                j+= 1

Example

Input: arr[]={8, 9, 0, 1, 2, 0, 3}
Output: arr[]={8, 9, 1, 2, 3, 0, 0}
Explanation:
Swap {0 ,1} -> Resulting array {8, 9, 1, 0, 2, 0, 3}
Swap {0 ,2} -> Resulting array {8, 9, 1, 2, 0, 0, 3}
Swap {0 ,3} -> Final array {8, 9, 1, 2, 3, 0, 0}

Input: arr[]={2, 5, 7, 0, 4, 0, 7, -5, 8, 0}
Output: arr[]={2, 5, 7, 4, 7, -5, 8, 0 ,0 ,0 }

In the example array [2, 5, 7, 0, 4, 0, 7, -5, 8, 0], a two-pointer technique is used where one pointer (right) scans the array, and the other pointer (left) tracks where to place the next non-zero element.
As right encounters non-zero elements, they are swapped with elements at the left pointer, which is then incremented, resulting in non-zero elements moving to the front while zeroes shift to the end. After completing the scan, the modified array is [2, 5, 7, 4, 7, -5, 8, 0, 0, 0].
