Trapping Rain Water

Introduction

Trapping Rain Water Problem states that given an array of N non-negative integers arr representing an elevation map where the width of each bar is 1, compute how much water it can trap after rain.
The Two-Pointer technique is a powerful approach for solving the Trapping Rain Water problem efficiently. By moving the pointers inward from both ends and keeping track of the maximum heights, we can calculate the trapped water without needing additional space, making this approach both time and space efficient.

Step By Step Explanation

  1. Take two pointers l and r. Initialize l to the starting index 0 and r to the last index N-1.
  2. Since l is the first element, left_max would be 0, and right_max for r would be 0.
  3. While l != r, iterate the array.
  4. We have two possible conditions:
    • Condition 1: left_max <= right_max
      1. Consider the element at index l.
      2. Since we have traversed all elements to the left of l, left_max is known.
      3. For the right max of l, we can say that the right max would always be >= current r_max here.
      4. So, min(left_max, right_max) would always equal to left_max in this case.
      5. Increment l.
    • Condition 2: left_max > right_max
      1. Consider the element at index r.
      2. Since we have traversed all elements to the right of r, right_max is known.
      3. For the left max of l, we can say that the left max would always be >= current l_max here.
      4. So, min(left_max, right_max) would always equal to right_max in this case.
      5. Decrement r.

This approach ensures the algorithm runs in linear time, O(n), since the array is traversed once.

Python Code

            
def trapping_rain_water(A):
    i, j = 0, len(A) - 1
    left_max, right_max = A[i], A[j]
    res = 0
    
    while i < j:
        left_max = max(left_max, A[i])
        right_max = max(right_max, A[j])
        
        if left_max < right_max:
            res += left_max - A[i]
            i += 1
        else:
            res += right_max - A[j]
            j -= 1
            
    return res

Example

Output: 6
